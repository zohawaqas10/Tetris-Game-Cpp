# Tetris Game in C++

## Overview
This is a **console-based Tetris game** developed in **C++** as a **semester project**.  
The game replicates classic Tetris gameplay: different shaped blocks fall from the top, and the player must arrange them to complete horizontal lines.  

The project was created using **Microsoft Visual Studio**.

---

## Technologies Used
- **Language:** C++  
- **IDE:** Microsoft Visual Studio  
- **Platform:** Windows (Console Application)  

---

## Features
- Classic Tetris gameplay  
- Randomly generated blocks  
- Real-time block movement  
- Block rotation  
- Line completion and clearing  
- Score tracking  
- Game over detection  
- Keyboard controls  

---

## Controls
| Key | Action |
| --- | ------ |
| ⬅️ Left Arrow | Move block left |
| ➡️ Right Arrow | Move block right |
| ⬇️ Down Arrow | Move block down faster |
| ⬆️ Up Arrow | Rotate block |
| ESC | Exit game |

---

## Game Logic
- Blocks fall from the top of the board  
- Player arranges blocks to form complete horizontal lines  
- Completed lines are cleared and score increases  
- Game ends when blocks reach the top  

---

## How to Run
1. Open **Microsoft Visual Studio**  
2. Click **Open a Project or Solution**  
3. Select the `.sln` file in the project folder  
4. Build the project (`Ctrl + Shift + B`)  
5. Run the project (`Ctrl + F5`)
