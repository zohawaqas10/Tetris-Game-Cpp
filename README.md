###Tetris Game in C++
##Project Overview

This project is a console-based Tetris game developed in C++ as a semester task.
The game follows the classic Tetris rules, where different shaped blocks fall from the top, and the player must arrange them efficiently to complete horizontal lines.

The project was developed and tested using Microsoft Visual Studio.

##Technologies Used

Programming Language: C++

IDE: Microsoft Visual Studio

Platform: Windows (Console Application)

##Features

Classic Tetris gameplay
Randomly generated Tetris blocks
Real-time block movement
Block rotation
Line completion and clearing
Score tracking
Game over detection
Keyboard-based controls

##Controls
Key	Action
⬅️ Left Arrow	Move block left
➡️ Right Arrow	Move block right
⬇️ Down Arrow	Move block down faster
⬆️ Up Arrow	Rotate block
ESC	Exit game


##Game Logic

Blocks fall from the top of the game board
Player arranges blocks to form complete horizontal lines
Completed lines are removed and score is increased
The game ends when blocks reach the top of the board

##How to Run the Project

Open Microsoft Visual Studio
Click Open a Project or Solution
Select the .sln file
Build the project (Ctrl + Shift + B)
Run the project (Ctrl + F5)
